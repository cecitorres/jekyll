## Indispensables
Si deseas saber que versión es tu sistema operativo.
```
lsb_release -a
```

Correr un script (.sh)

```
sh file.sh
# O también
bash file.sh
```

Recuerda que debe tener permisos para ejecutarse (chmod +x script-name-here.sh)
